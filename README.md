# time-wizard

> A time tracking terminal application

## Install

Using yarn:

```sh
$ yarn global add time-wizard
```

Using npm:

```sh
$ npm install -g time-wizard
```

## Usage

```sh
$ time-wizard --help

  Usage: time-wizard [options] [command]

  Options:

    -h, --help                  output usage information

  Commands:

    start <project> [tasks...]
    stop
    restart
    status
    report [options]
    projects
```
